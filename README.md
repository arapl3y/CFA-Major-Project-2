### Installation

```sh
$ git clone https://github.com/arapl3y/CFA-Major-Project-2.git
$ cd ../CFA-Major-Project-2/
$ bundle install
$ rails db:create
$ rails db:migrate
$ rails server
```
Then navigate to http://localhost:3000 in the browser.
